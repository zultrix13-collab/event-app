# Roadmap (Starter)

This roadmap follows the sequence in `MarTech-MVP-v1-Spec.md`.

## Phase outline

1. Project bootstrap
2. Auth + organization foundation
3. Plans + subscription foundation
4. Meta OAuth integration
5. Sync jobs pipeline
6. Metrics storage + dashboard
7. AI recommendation engine
8. QPay billing
9. Observability + admin tools
10. Reliability hardening
11. QA + release readiness

## Working rules

- Build only one phase at a time.
- Keep migrations and docs synchronized with implementation.
- Avoid implementing business logic outside the active phase.
